# 🃏 CARD FLIP UI

An interactive playing cards interface built using pure HTML and CSS.
--
## 🚀 Features
- Smooth hover animations (lift + scale)
- 3D flip effect using CSS transforms
- Clean and responsive card layout
- Reusable component-based structure



## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Transforms, Transitions)



## 🎯 What I Learned
- Structuring components using nested divs
- Using `transform`, `perspective`, and `rotateY`
- Separating container and animation layers
- Creating interactive UI without JavaScript


## 🔮 Future Improvements
- Add JavaScript for click-to-flip
- Shuffle and random card generation
- Sound effects for interaction
- Convert into a mini card game
